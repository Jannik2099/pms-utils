Slot
====
.. currentmodule:: pms_utils.atom

This class combines:

* a package slot as per `PMS 3.1.3 <https://projects.gentoo.org/pms/8/pms.html#x1-190003.1.3>`_
* a dependency subslot as per `PMS 8.3.3 SUB-SLOT <https://projects.gentoo.org/pms/8/pms.html#x1-820008.3.3>`_

The subtypes for slot / subslot are not exposed in the Python bindings as of now.
